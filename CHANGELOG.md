## [1.0.0] - 2026-01-XX

### Added
- Initial release
- 71 political dynasties tracked
- Interactive network visualization
- Timeline of 2025 events (49 events)
- Regional analysis by province
- Corruption case tracker (POGO focus)
- Weekly review system
- Data validation scripts
- GitHub Pages deployment

### Data
- 71 major dynasties with relationships
- 148+ key politicians tracked
- 71 provinces mapped
- Regional breakdown: 37 Luzon, 17 Visayas, 18 Mindanao

### Features
- Interactive force-directed network graph
- Timeline visualization with filtering
- Regional charts and statistics
- Corruption case tracking
- Weekly news digests
